¿Por qué es importante que los proyectos de software mantengan 
un registro de cambios usando herramientas como Git?

Es importante que los proyectos de software mantengan un registro 
de cambios usando herramientas como Git porque permite llevar 
un control detallado de todas las modificaciones realizadas en el código. Además, 
Git permite recuperar versiones anteriores del software en caso 
de errores o problemas. Si una actualización introduce fallos, es posible 
volver a una versión estable sin perder el trabajo realizado. 
Esto aumenta la seguridad y la confiabilidad durante el desarrollo.
